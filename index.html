External Calendars

Read-only .ics feeds (sports schedules, school district calendars, etc.) shown alongside your notes. Fetched directly from the browser; if a feed blocks that, a public CORS-proxy is used as a fallback.

Google Calendar Sync

Publishes your day notes (and A/B-Day labels) as a live .ics feed via a secret GitHub Gist, which Google Calendar can subscribe to. The gist URL is unauthenticated once created — anyone with the link can read it, so keep it as private as the token below.
